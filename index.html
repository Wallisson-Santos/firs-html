História dos computadores

(Voltar ao início)

A história dos computadores começou com dispositivos de cálculo simples, como o ábaco, máquina analítica, considerada um precursor dos computadores modernos e hoje, a tecnologia avançou tanto que foca em inteligência artificial e computação em nuvem, enquanto a pesquisa em computação quântica promete resolver problemas complexos utilizando princípios da física quântica. Essa é a evolução fascinante dos computadores ao longo dos séculos.

Cliente x Servidores

(Voltar ao início)

É o dispositivo ou software que faz uma solicitação de serviço ou recurso. Servidor é o dispositivo ou software que responde às solicitações dos clientes, fornecendo os serviços ou recursos solicitados.

Estutura básica do HTML

(Voltar ao início)

O HTML (HyperText Markup Language) é uma linguagem de marcação usada para criar páginas. Na página HTML contém os seguintes elemetos:

  • <!DOCTYPE html>: Declaração que define o tipo de documento e a versão do HTML.
  • <html>: Elemento raiz que envolve todo o conteúdo da página.
  • <head>: Contém metadados sobre o documento, como o título, links para folhas de estilo e scripts.
  • <body>: Contém o conteúdo visível da página, como textos, imagens, links e outros elementos.

Sobre as Tags

(Voltar ao início)

As tags são essenciais para criar e organizar o conteúdo de uma página web de forma estruturada e acessível. Cada tag tem uma função específica e é representada por um nome entre colchetes angulares, como <tag>.

  • Formatação de Texto:

    • <h1> a <h6>: definem o tamanho dos títulos;
    • <p>: define os parágrafos;
    • <span>: estiliza partes do texto;

  • Links e Imagens:

    • <a>: cria um hyperlink para outra página ou recurso.;
    • <span>: estiliza partes do texto.

  • Listas:

    • <ul>: cria uma lista não ordenada com marcadores;
    • <li>: cria uma lista ordenada numerada
    • <li>cria um item da lista;
